<summary>âœ… How do I add Dartâ€™s global bin to PATH?</summary>

### macOS / Linux

Add this line to your shell config file (`~/.zshrc`, `~/.bashrc`, etc.):

```sh

```

Then reload your terminal:

```sh

```

